Differences between Block and Inline Element

BLOCK ELEMENT Vs INLINE ELEMENT

Block Level Element Inline Level Element
Begins on a new line of text. Does not begin on a new line of text. Text is placed on the same line.
Its width extends beyond the inner height values. Its width only extends as far as the inner content.
You can set the width and height values. You can't set the width and height values.
Can contain text, data, inline elements, or other block level element. Can contain text, data, or other inline element.
